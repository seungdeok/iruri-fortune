# PRD: iruri-fortune (오늘의 운세)

## 목표
생년월일 + 양/음력만 입력하면 오늘 날짜와 조합해 **클라이언트에서 결정적으로 계산한 오늘의 운세**를 한 화면에 보여줘요. 백엔드 없이 동작하고, 같은 사람·같은 날에는 항상 같은 결과를 하루 종일 유지해요.

## 사용자
토스 앱 안에서 가볍게 오늘의 운세를 확인하려는 사람. 로그인·회원가입 없이 생년월일만 한 번 입력하면 다음날부터는 바로 오늘의 운세를 봐요.

## 핵심 기능
1. 생년월일·양음력 입력 + 기기 저장 (앱인토스 `Storage` 우선, 없으면 `localStorage` 폴백 — 다음날 재입력 불필요)
2. 띠(12지) 자동 산출 + 총운 (점수 · 등급 · 별점 · 한 줄 메시지)
3. 분야별 운세 4종(애정·금전·건강·직장) + 행운의 색 · 행운의 숫자

## MVP 제외 사항
- 별도 백엔드 / API / DB
- 로그인 · 회원가입 · 사용자 계정
- 다중 라우트 · 네비게이션 스택
- 정밀 음↔양력 변환 (설날 경계 정확도) — 띠는 생년으로 산출, 양/음력은 운세 시드에만 반영
- 운세 공유 · 푸시 · 히스토리(과거 날짜 운세 조회)
- AI/LLM 기반 실시간 운세 문구 생성 — 문구는 큐레이션 풀에서 선택

## 디자인
- **다크/라이트**: TDS Web(`@toss/tds-mobile`) 컴포넌트만 사용, 토스 기본 테마를 따름
- **색상**: primaryColor `#3FD599` (`granite.config.ts`) + TDS 무채색 기반
- **UX 라이팅**: 한국어 해요체 (TDS UX 라이팅 가이드 준수)
- **레이아웃**: 라우터 없는 단일 화면. 입력 상태 ↔ 결과 상태를 같은 화면에서 전환

## 인수 조건 (Acceptance)
- [ ] 첫 진입 시 생년월일 입력 화면이 보이고 양력/음력을 선택할 수 있다
- [ ] '오늘의 운세 보기'를 누르면 같은 화면이 결과 뷰로 전환된다
- [ ] 결과 화면에 총운(점수 + 별점 + 한 줄 메시지)이 표시된다
- [ ] 결과 화면에 분야별 운세 4종(애정·금전·건강·직장)이 점수/코멘트와 함께 표시된다
- [ ] 결과 화면에 행운의 아이템(행운의 색, 행운의 숫자)이 표시된다
- [ ] 입력한 생년월일로부터 띠(12지)가 자동 산출되어 노출된다
- [ ] 같은 생년월일로 같은 날 여러 번 열어도 운세 결과가 완전히 동일하다 (결정적)
- [ ] 앱을 껐다 켜도 생년월일이 유지되어 바로 오늘의 운세가 보인다
- [ ] 날짜가 바뀌면 같은 생년월일에 대해 새로운 운세가 계산된다
- [ ] 생년월일을 수정할 수 있는 진입점('생일 다시 입력')이 있다
- [ ] 전 화면이 TDS Web 컴포넌트로 구성되고 `pnpm dev`로 정상 렌더링된다
- [ ] `pnpm lint` 통과, 타입 에러 없음
